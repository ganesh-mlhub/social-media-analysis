# 📊 Social Media Analytics Dashboard

## 🚀 Project Overview
This project is a full-stack web application that analyzes social media posts and displays insights using charts.

It includes:
- Backend (Node.js + Express)
- Database (MongoDB)
- Frontend (React + Chart.js)

---

## 🎯 Features

✅ Add social media posts  
✅ Sentiment analysis on comments  
✅ Trending hashtags  
✅ Active users analysis  
✅ Likes distribution  
✅ Interactive charts (Bar + Pie)

---

## 🛠️ Tech Stack

### Frontend
- React.js
- Chart.js
- Axios

### Backend
- Node.js
- Express.js

### Database
- MongoDB (Local / Atlas)

## 🌐 API Endpoints

| Method | Endpoint        | Description              |
|--------|---------------|--------------------------|
| POST   | /addPost      | Add posts                |
| GET    | /trending     | Get trending hashtags    |
| GET    | /active-users | Get active users         |
| GET    | /likes        | Get likes analytics      |
